# IGDM Universal - 快速开始指南

## 🎉 恭喜！项目已创建成功

您现在拥有一个跨平台的 Instagram DM 管理应用！

---

## 🚀 立即运行

### 方法 1：开发模式（推荐测试）

```bash
cd C:\Users\user\Desktop\igdm-universal
npm run electron:dev
```

这将同时启动：
- ✅ Vite 开发服务器（前端）
- ✅ Electron 应用（桌面窗口）

**第一次运行需要等待约 10-20 秒**

---

### 方法 2：Web 版本（浏览器测试）

```bash
cd C:\Users\user\Desktop\igdm-universal
npm run dev
```

然后在浏览器打开：http://localhost:5173

## 📦 打包成安装程序

### Windows 安装包

```bash
npm run electron:build-win
```

输出：
- `dist-electron/IGDM Universal Setup 1.0.0.exe` - 安装版
- `dist-electron/IGDM Universal 1.0.0.exe` - 便携版

### 全平台打包

```bash
npm run electron:build
```

## 🎯 功能说明

### 1. 多账户管理

**添加账户**：
1. 点击右上角账户按钮
2. 选择"添加账户"
3. 输入账户名称（如："个人账户"、"工作账户"）
4. 在 Instagram 页面登录

**切换账户**：
- 点击当前账户名
- 选择要切换的账户

**删除账户**：
- 账户列表中点击 × 按钮

### 2. 桌面通知

应用会自动请求通知权限，允许后：
- ✅ 新消息会显示桌面通知
- ✅ 显示未读消息数量

### 3. 快捷键

- `Ctrl/Cmd + R` - 刷新页面
- `Ctrl/Cmd + Q` - 退出应用

## 📱 添加手机版支持

想要生成 Android/iOS 版本？

### 安装 Capacitor

```bash
npm install @capacitor/core @capacitor/cli
npx cap init igdm com.igdm.app --web-dir=dist
```

### 添加平台

```bash
# Android
npx cap add android
npm run build
npx cap sync
npx cap open android

# iOS (需要 macOS)
npx cap add ios
npm run build
npx cap sync
npx cap open ios
```

## 💡 常见问题

### Q: 应用无法启动？
A: 确保已安装依赖：`npm install`

### Q: Instagram 无法登录？
A: 这是正常的 Instagram Web 登录，使用您的账户和密码登录即可

### Q: 如何更新应用？
A: 修改代码后，重新运行 `npm run electron:dev` 或重新打包

### Q: 可以同时运行多个窗口吗？
A: 目前设计为单窗口，多账户通过切换功能使用

### Q: 数据存储在哪里？
A: 账户信息存储在浏览器 LocalStorage 中，不会上传到服务器

## 📊 性能优化建议

1. **减少内存占用**
   - 关闭不用的账户
   - 定期清理浏览器缓存

2. **提高启动速度**
   - 使用生产版本（打包后的应用）
   - 减少开机自启动程序

3. **网络优化**
   - 使用稳定的网络连接
   - 考虑使用代理（如需要）
